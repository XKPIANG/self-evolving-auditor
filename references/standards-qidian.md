# 起点小说发布标准

## 平台特征

起点中文网是阅文集团旗下付费阅读平台，网文行业标杆。用户画像：
- 主力用户：20-40岁，核心网文读者，有付费习惯
- 阅读习惯：接受较长篇幅，重视世界观和角色塑造，愿意为好内容付费
- 付费模式：VIP章节付费（千字5分），月票/推荐票打榜
- 核心指标：收藏、推荐票、月票、订阅数、均订

## 一、硬性标准（必须达标）

### 1.1 字数要求
- 单章字数：2000-5000字（推荐3000-4000）
- 总字数：签约门槛通常5万字以上（30万字以上可申请完本奖）
- 日更要求：签约后日更4000-6000字，大神作者可适当放宽
- VIP章节：通常在20-30万字后上架

### 1.2 内容红线
- [ ] 无政治敏感内容
- [ ] 无色情/暴力过度描写（起点审核比番茄更严）
- [ ] 无涉政/涉军/涉警敏感题材
- [ ] 无抄袭/洗稿嫌疑
- [ ] 无真实人物/事件影射
- [ ] 无三观不正的明显引导
- [ ] 宗教/民族/历史相关内容需谨慎

### 1.3 格式规范
- [ ] 章节标题格式统一（第X章 + 标题）
- [ ] 无乱码/特殊字符
- [ ] 段落分明，每段不超过200字
- [ ] 对话标记清晰（引号使用统一）
- [ ] 无多余空行/空格

---

## 二、开篇标准（前5章决定签约）

起点编辑看前5章决定是否签约。与番茄不同，起点允许稍慢的开局，但质量要求更高。

### 2.1 第1章必须做到
- [ ] 500-1000字内建立代入感（主角处境/性格/目标）
- [ ] 有明确的悬念或冲突牵引
- [ ] 文笔流畅，无明显病句/错字
- [ ] 不要大段设定介绍，但世界观可以自然融入

### 2.2 前5章必须做到
- [ ] 主角性格鲜明（不只是"普通人穿越"）
- [ ] 核心金手指/能力体系初步展现
- [ ] 至少1条主线冲突建立
- [ ] 世界观通过情节自然展开（不要说明文）
- [ ] 至少1个让人想继续看的钩子
- [ ] 文笔质量在线（起点读者对文笔有要求）

### 2.3 开篇禁忌
- ❌ 大段景物描写开头
- ❌ 前3章全是设定堆砌
- ❌ 主角性格模糊（"路人脸"主角）
- ❌ 文笔粗糙（病句、错字、口水话）
- ❌ 模仿痕迹过重（起点编辑一眼能看出）

---

## 三、章节结构标准

### 3.1 章节开头
- [ ] 自然衔接上一章（不要生硬跳转）
- [ ] 前200字要有事件推进或情绪变化
- [ ] 可以适当慢热，但不能无聊

### 3.2 章节中段
- [ ] 推进剧情、深化角色、展开世界观三者至少占一
- [ ] 信息密度适中（不要注水也不要太赶）
- [ ] 对话/动作/心理/描写四种手法交替使用
- [ ] 适当留白，给读者想象空间

### 3.3 章节结尾
- [ ] 必须有钩子（悬念/反转/新信息/情绪转折）
- [ ] 不要在平淡处结束
- [ ] 高潮章可以戛然而止（制造追读欲）
- [ ] 日常章结尾要有过渡到下一事件的引子

### 3.4 章节字数控制
| 类型 | 推荐字数 | 说明 |
|------|----------|------|
| 日常章 | 3000-3500 | 保持信息量，不要纯水 |
| 冲突章 | 3500-4500 | 需要篇幅展开对抗 |
| 高潮章 | 4000-5000 | 允许写长，节奏紧凑 |
| 过渡章 | 2000-3000 | 尽量短，但要有信息量 |
| VIP章 | 3000-5000 | 付费内容要有足够信息量 |

## 四、付费留存优化

起点靠付费章节变现，VIP上架后的订阅数是核心指标。

### 4.1 上架时机
- 通常20-30万字上架VIP
- 上架前要积累足够收藏（通常3000+收藏才有意义）
- 上架章节要是高潮章或关键转折（不能在平淡处上架）
- 上架前1-2章要制造强烈追读欲

### 4.2 均订优化
- 均订 = 总订阅章节数 / VIP章节数
- 保持更新频率稳定（断更是均订杀手）
- 定期释放大爽点（5-8章一个）
- 保持主角成长线清晰（读者追的是成长感）

### 4.3 月票/推荐票优化
- 制造投票动力（月票加更、推荐票加更）
- 月底是月票战关键期，需要大高潮配合
- 与读者互动（章末求票、回复书评）

### 4.4 养书率优化
- 养书 = 读者攒着不看，等多了再看
- 养书率高说明节奏太慢或更新太慢
- 对策：加快节奏、增加日更量、制造"不看就亏"的紧迫感

## 五、题材热度参考

起点热门题材（2025-2026）：
- 玄幻：东方玄幻、高武世界、诸天万界
- 科幻：星际文明、末日废土、赛博朋克
- 仙侠：修仙文明、凡人修仙、洪荒
- 都市：重生、系统、异能
- 历史：架空历史、穿越历史、三国
- 游戏：虚拟现实、游戏制作、电竞
- 无限流：副本、规则怪谈、惊悚

### 起点 vs 番茄题材差异
| 维度 | 起点 | 番茄 |
|------|------|------|
| 主流题材 | 玄幻、科幻、仙侠 | 都市、赘婿、重生 |
| 文风要求 | 较高，允许慢热 | 快节奏，必须抓人 |
| 世界观 | 鼓励宏大设定 | 简单直接 |
| 角色塑造 | 要求立体 | 可以脸谱化 |
| 字数 | 长篇为主（200万+） | 中短篇也可 |

## 六、发布前检查清单

### 必查项
- [ ] 前5章文笔流畅
- [ ] 前5章主角性格鲜明
- [ ] 前5章有核心冲突
- [ ] 世界观通过情节展开（非说明文）
- [ ] 每章结尾有钩子
- [ ] 无内容红线问题
- [ ] 格式统一
- [ ] 无错别字/病句
- [ ] 章节字数在合理范围

### 推荐项
- [ ] 书名有辨识度（2-6字，有记忆点）
- [ ] 简介精准（100字内说清核心卖点+主角特点）
- [ ] 标签准确（影响推荐精准度）
- [ ] 封面专业（起点提供封面制作工具）
- [ ] 有大纲（编辑会看）

---

## 发布水平评分标准（10分制）

| 分数 | 标准 |
|------|------|
| 9-10 | 完全达到起点签约标准，文笔出色，可冲新书榜 |
| 7-8 | 基本达标，文笔流畅，微调后可签约 |
| 5-6 | 部分达标，需要提升文笔或结构调整 |
| 3-4 | 多项不达标，需要较大改动 |
| 1-2 | 不建议在起点发布 |
