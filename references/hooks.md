# 爽点设计模板

## 什么是爽点

爽点 = 读者期待的满足。核心公式：

**铺垫（制造期待）→ 释放（满足期待）→ 余韵（超预期满足）**

没有铺垫的爽是空爽。铺垫太长的爽是憋屈。铺垫精准的爽是上瘾。

## 番茄热门爽点类型

### 1. 打脸型

**结构：** 被小看/嘲讽 → 展示实力 → 围观震惊

**模板：**
```
[反派/配角] 看不起主角，当众[嘲讽/挑衅/下判断]
↓ 主角不动声色
↓ 主角展示[能力/身份/实力]
↓ 围观者震惊，反派脸色大变
↓ [可选] 更高层人物注意到主角
```

**铺垫长度：** 1-3章（不宜超过5章，否则读者会觉得憋屈太久）

**变体：**
- 装猪吃虎型：主动隐藏实力被打脸后翻盘
- 身份反转型：以为是废物其实是天才
- 以弱胜强型：所有人都不看好结果赢了

---

### 2. 升级/突破型

**结构：** 修炼/积累 → 突破 → 实力碾压

**模板：**
```
主角[修炼/研究/吸收]了[时间]
↓ 遇到瓶颈或关键时刻
↓ [可选] 外部压力（敌人逼近/时限将至）
↓ 突破！
↓ 实力质变，之前打不过的现在轻松碾压
```

**铺垫长度：** 5-15章（需要足够的积累感）

**关键：** 突破前后要有明确的实力对比参照物

---

### 3. 身份揭露型

**结构：** 隐藏身份 → 被迫/主动揭露 → 围观震惊

**模板：**
```
主角一直以[低调身份]示人
↓ 某个关键时刻不得不暴露
↓ [或] 被第三方揭穿
↓ 所有人震惊：原来他就是[真实身份]！
↓ 态度180度转变
```

**铺垫长度：** 可长可短，但需要至少1-3次"差点被发现"的悬念

---

### 4. 复仇型

**结构：** 被欺负/被害 → 隐忍/成长 → 反杀

**模板：**
```
[反派] 对主角做了[过分的事]
↓ 主角[隐忍/逃脱/成长]
↓ 实力/资源足够
↓ 正面碾压/算计反杀
↓ 反派绝望
```

**铺垫长度：** 越长越爽，但需要中间穿插小爽点维持耐心

---

### 5. 宝物/机缘型

**结构：** 发现 → 获取 → 获得能力

**模板：**
```
主角发现[宝物/秘境/传承]的线索
↓ [可选] 多方势力争夺
↓ 主角凭借[智慧/运气/特殊条件]获得
↓ 实力/资源大幅提升
↓ 其他人后悔/眼红
```

---

### 6. 收小弟/建势力型

**结构：** 展示实力 → 人才投靠 → 势力壮大

**模板：**
```
主角展示[能力/魅力/资源]
↓ [强者/天才] 主动投靠
↓ 势力壮大
↓ [可选] 原来瞧不起的人求着加入
```

---

### 7. 打赌/比试型

**结构：** 被挑战 → 约定赌注 → 赢得赌局

**模板：**
```
[对手] 公开挑战/打赌
↓ 赌注对主角不利
↓ 所有人都不看好主角
↓ 比试过程（可以先落后再翻盘）
↓ 主角赢，获得赌注
↓ 对手付出代价
```

---

### 8. 装逼型（被动装逼）

**结构：** 主角做了某事 → 没当回事 → 别人震惊

**模板：**
```
主角[随手做了某件事]
↓ 主角觉得没什么大不了
↓ 旁观者/专家/高层发现并震惊
↓ "这怎么可能！""他居然……"
↓ 主角身份/实力被重新评估
```

**关键：** 主角越淡定，旁观者越震惊，效果越好

---

## 爽点组合技

高明的写法是多个爽点叠加：

```
打脸 + 升级：被嘲笑→突破→碾压嘲笑者
身份揭露 + 打脸：暴露真实身份→打脸所有质疑者
复仇 + 装逼：轻松反杀曾经的强敌→旁人震惊
```

## 爽点评分标准（10分制）

| 分数 | 标准 |
|------|------|
| 9-10 | 铺垫精准，释放痛快，余韵悠长，读者想截图分享 |
| 7-8 | 爽点到位，但铺垫略长或释放不够彻底 |
| 5-6 | 有爽点但力度不够，或铺垫太长导致爽感被稀释 |
| 3-4 | 爽点模糊，读者感觉"就这？" |
| 1-2 | 没有爽点，或爽点和铺垫完全不匹配 |

## 爽点密度建议（番茄）

| 章节类型 | 爽点密度 |
|----------|----------|
| 开篇前3章 | 至少1个强爽点 |
| 日常章节 | 每3-5章1个中等爽点 |
| 高潮章节 | 1-2个强爽点叠加 |
| 过渡章节 | 至少1个小爽点（不能完全空白）|

## 起点小说爽点特点

### 起点 vs 番茄爽点差异

| 维度 | 番茄 | 起点 |
|------|------|------|
| 爽点类型 | 直接、粗暴、快感优先 | 有铺垫的爽、智商在线的爽 |
| 铺垫长度 | 1-3章 | 3-10章（允许更长铺垫） |
| 释放方式 | 打脸碾压为主 | 成长+打脸+智谋组合 |
| 读者口味 | 越爽越好 | 爽的同时要有逻辑 |
| 日常章 | 尽量减少 | 有存在价值（塑造角色） |

### 起点特色爽点类型

#### 9. 智谋布局型（起点偏好）

**结构：** 布局 → 对手入套 → 真相揭晓 → 围观震惊

**模板：**
```
主角提前[布局/设局/预判]
↓ 对手以为自己赢了
↓ 关键时刻真相揭晓
↓ 所有人发现主角早已算好一切
↓ "原来他早就……"
```

**关键：** 起点读者喜欢看"聪明的主角"，智谋型爽点比纯力量碾压更受欢迎

#### 10. 世界观揭秘型（起点偏好）

**结构：** 埋下伏笔 → 逐步揭开 → 真相震撼 → 格局提升

**模板：**
```
前期埋下[伏笔/疑问]
↓ 读者以为自己猜到了
↓ 真相比猜测更宏大
↓ 世界观格局提升到新层次
↓ "原来这个世界……"
```

**关键：** 起点读者喜欢宏大世界观，揭秘型爽点是长篇利器

#### 11. 角色成长型（起点偏好）

**结构：** 菜鸟 → 磨练 → 蜕变 → 独当一面

**模板：**
```
主角初期[弱小/迷茫/被低估]
↓ 经历一系列考验和磨练
↓ 逐渐掌握能力/建立信念
↓ 在关键时刻独当一面
↓ 读者见证整个成长过程的满足感
```

**关键：** 起点读者更看重成长过程，不是一步到位的爽
