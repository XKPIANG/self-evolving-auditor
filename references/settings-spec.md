## 设定一致性检查流程

审核新章节时，自动执行：

### 1. 规则违背检查

```
读取 settings.md 中所有"生效"的规则
扫描新章节内容
发现与规则矛盾的描述 → 报错

示例：
  S001: "这个世界没有魔法"
  新章节: "张三念了一段咒语，一道火球飞出"
  → 报错: 违反世界观规则S001（这个世界没有魔法），除非规则已废弃或修改
```

### 2. 能力体系检查

```
读取能力规则和限制
检查新章节中的能力使用是否符合

示例：
  A001: "火球术，炼气二层可学"
  characters.md: 张三当前 = 炼气一层
  新章节: "张三使出了火球术"
  → 报错: 张三当前炼气一层，不满足火球术学习条件（需炼气二层）

示例：
  AL001: "灵力耗尽后无法使用法术"
  新章节: 张三灵力耗尽后仍然放了三个火球
  → 报错: 违反能力限制AL001
```

### 3. 势力关系检查

```
读取势力关系表
检查新章节中的势力互动是否矛盾

示例：
  青云宗-天剑宗 = 敌对
  新章节: "青云宗和天剑宗联合举办大会"
  → 警告: 两宗关系为敌对，联合办会需要剧情解释
```

### 4. 地理一致性检查

```
读取地理设定
检查距离、位置描述是否矛盾

示例：
  黑风谷: "青云宗北100里"
  新章节: "张三从青云宗出发，走了半个时辰就到了黑风谷"
  → 警告: 100里路程半个时辰（1小时）到达，速度是否合理？
```

### 5. 输出设定一致性报告

```markdown
# 设定一致性检查 — 《书名》第X章

## ✅ 通过
- 能力使用符合等级要求
- 势力关系无矛盾
- 地理描述一致

## ⚠️ 警告
- 张三灵力已3章未恢复，建议补充恢复情节
- 天剑宗在第10章后消失已久，是否有意为之？

## ❌ 错误
- [严重] 违反规则S001：本章出现魔法描写，但世界观设定"没有魔法"
  - 规则确立: 第1章 "这个世界从来没有什么魔法"
  - 本章: "张三念了一段咒语，火球飞出"
  - 建议: 修改为武技描写，或在前面章节补充魔法设定的伏笔

- [严重] 能力不符：张三使用火球术，但当前炼气一层（需炼气二层）
  - 人物记录: 炼气一层（第5章未突破）
  - 能力要求: A001 火球术需炼气二层
  - 建议: 先安排突破情节，或改用其他技能
```
