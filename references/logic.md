# 逻辑检查清单

## 核心原则

逻辑问题 = 读者出戏。一次出戏需要十个爽点才能拉回来。

## 一、世界观逻辑

### 1.1 设定一致性
- [ ] 世界观规则有没有自相矛盾？
- [ ] 同一事物在不同章节的描述是否一致？
- [ ] 规则有没有被作者自己打破？（比如"这个世界没有魔法"然后角色用了魔法）

### 1.2 金手指/能力体系
- [ ] 能力获取逻辑是否自洽？（为什么主角有这个能力？）
- [ ] 能力上限是否明确？（避免"需要的时候突然变强"）
- [ ] 能力代价是否合理？（无代价的强能力=无张力）
- [ ] 有没有"上次能用这次不能用"的情况？

### 1.3 势力/组织设定
- [ ] 势力之间的关系是否清晰？
- [ ] 势力的实力等级是否合理？
- [ ] 组织的运作逻辑是否通顺？（一个门派不可能只有掌门和弟子两个人）

### 1.4 时间线
- [ ] 事件发生顺序是否合理？
- [ ] 时间跨度是否和字数匹配？（3000字写了三个月的事=流水账）
- [ ] 有没有时间矛盾？（前面说三天后，后面变成了一天后）

---

## 二、角色逻辑

### 2.1 动机合理性
- [ ] 角色做每件事的动机是否充分？
- [ ] 有没有"为了剧情需要"而做出不合理行为的角色？
- [ ] 反派的动机是否立体？（纯坏=纸片人）

### 2.2 性格一致性
- [ ] 角色性格是否前后一致？
- [ ] 性格转变是否有铺垫和过渡？
- [ ] 角色的说话方式是否符合其身份/背景？

### 2.3 智商在线
- [ ] 角色是否做出了符合其智商的决策？
- [ ] 有没有"降智"行为？（聪明人突然犯蠢推动剧情）
- [ ] 反派是否太蠢？（太蠢的反派会降低爽感）

### 2.4 行为合理性
- [ ] 角色在特定情境下的反应是否合理？
- [ ] 有没有"正常人不会这么做"的行为？
- [ ] 角色的记忆是否连贯？（不该知道的事突然知道了）

---

## 三、剧情逻辑

### 3.1 因果链
- [ ] 每个事件的原因是否充分？
- [ ] 有没有"突然发生"的事件？（无因无果）
- [ ] 前后事件的因果关系是否通顺？

### 3.2 伏笔管理
- [ ] 每个伏笔是否都有回收？
- [ ] 伏笔回收的时间间隔是否合理？
- [ ] 有没有"突然想起来"的伏笔？（太刻意）

### 3.3 冲突合理性
- [ ] 冲突的起因是否合理？
- [ ] 冲突的升级是否自然？
- [ ] 冲突的解决是否令人满意？（不能突然天降神兵）

### 3.4 节奏与逻辑的平衡
- [ ] 有没有为了赶节奏而跳过必要逻辑？
- [ ] 有没有为了制造冲突而强行降智？
- [ ] 有没有为了爽感而牺牲合理性？

---

## 四、细节逻辑

### 4.1 战斗/对抗场景
- [ ] 战斗过程是否符合能力设定？
- [ ] 伤势描写是否合理？（中了一刀还能打三章=出戏）
- [ ] 战斗结果是否和实力对比匹配？

### 4.2 日常场景
- [ ] 人物对话是否自然？
- [ ] 生活细节是否合理？（古代人用手机这种低级错误）
- [ ] 场景描写是否连贯？

### 4.3 数字/数据
- [ ] 金额、距离、时间等数字是否前后一致？
- [ ] 有没有明显的数字矛盾？（前面说100人，后面突然变成1000人）
- [ ] 数字是否符合常识？

---

## 五、常见逻辑硬伤

### 5.1 "主角光环"过度
症状：所有巧合都对主角有利，所有敌人都刚好犯蠢
危害：降低读者代入感和爽感
修复：给主角制造真正的困境，让胜利有代价

### 5.2 "工具人"配角
症状：配角只在需要的时候出现，用完就消失
危害：世界观单薄
修复：给配角独立的行为线

### 5.3 "设定膨胀"
症状：越写越强，设定越来越大，最后收不住
危害：后期崩盘
修复：早期就规划好能力上限

### 5.4 "记忆选择性失忆"
症状：角色选择性忘记重要信息
危害：严重出戏
修复：建立角色信息记录表

## 逻辑评分标准（10分制）

| 分数 | 标准 |
|------|------|
| 9-10 | 逻辑严密，设定自洽，伏笔回收完整，读者挑不出毛病 |
| 7-8 | 整体通顺，有1-2处小瑕疵但不影响阅读 |
| 5-6 | 基本通顺，但有明显逻辑漏洞或设定矛盾 |
| 3-4 | 逻辑混乱，多处硬伤，读者频繁出戏 |
| 1-2 | 基本没有逻辑可言 |
