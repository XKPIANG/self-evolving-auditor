# 章节目录

每章审核时自动生成 `chXXX.md` 文件。

## 文件格式

```markdown
# 第X章：[章节标题]

## 基本信息
- **字数：** XXXX
- **情绪值：** -5 到 +5
- **故事时间：**

## 内容摘要
[100-200字概括本章核心事件]

## 关键事件
- [事件1]
- [事件2]

## 人物出场
- 人物名：[本章状态变化]

## 数据变动
- 财产/等级/物品变动

## 钩子
- 埋下：[新钩子]
- 推进：[已有钩子进展]
- 解决：[已解决钩子]

## 伏笔
- 埋下：[新伏笔]
- 回收：[已回收伏笔]

## 审核备注
- 节奏：
- 爽点：
- 逻辑问题：
```
